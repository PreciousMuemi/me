# Cynthia Muemi - Professional Portfolio

<!-- ![Portfolio Screenshot](./images/profile.jpg) -->

A modern, responsive portfolio website showcasing Cynthia Muemi's work as a Smart Contract Developer and Blockchain Engineer.

## Features

- **Responsive Design**: Works on all device sizes
- **Dark/Light Mode**: Toggle between color schemes
- **Customizable Themes**: 5 different color themes
- **Interactive Elements**: Typewriter effect for roles
- **Multi-page Navigation**: 
  - About Me
  - Services
  - Portfolio
  - Contact
  - Skills

## Technologies Used

- HTML5
- CSS3

## Setup

1. Clone this repository
2. Open `index.html` in your browser
3. No build process or dependencies required
